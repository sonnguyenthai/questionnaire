INSTALL
--------------
- Require PHP >=5.5.9
- Composer 1.5.6
- Setup the project:
    ~~~
     $ git clone https://github.com/sonnguyenthai/questionnaire.git
     $ cd questionnaire
     $ composer install
    ~~~
- Enjoy!
